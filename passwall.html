File passwall Tải Xuống

Login vào openwrt 

Chọn System > Software > Update lists ( phải có mạng nha ) sau khi update xong > Upload Package > Browse > chọn thư mục vừa giải nén lúc nãy vào thư mục A > chọn file > open > Upload > Install

** làm lần lượt cho đến khi đã cài hết các file ở thư mục Anh

Sau khi đã cài xong hết các file thì cài luci-app-paswall ở thư mục Cài sau cuối cùng !

Cách Setting !

Chọn Services > passwall  > Other settings rồi setting như ảnh ấn Save


Tiếp đến là node list


Hiện ra như này các bạn điền Vmess /vless /trojan vào rồi ấn add 



add thàng công các bạn quay về mục Basic settings rồi cài như mình 



Ở Phần TCP Node chọn server bạn vừa add ở phần node list nhé !
