 Phát wifi thông qua VPN Non Root

Để phát wifi từ thiết bị này sang thiết bị khác thông qua kết nối chia sẻ wifi ( Hotspot )

Ta sẽ dựa trên nền phát wifi nền proxy


Ứng dụng cần tải là VPN Hotspot & Proxy

(Tuỳ vào điện thoại mà cách hiển thị có thể khác nhau)


Bước 1 : Vào Cài Đặt > Kết Nối > Chia sẻ kết nối internet > Điểm truy cập di động


Bước 2 : Cấu hình lại lại tên và mật khẩu chia sẻ


Bước 3 : Bật ứng dụng VPN lên và bật Điểm truy cập di động


Bước 4 : Vào ứng dụng VPN Hotspot & Proxy > Nhập Port 8080 > Start Proxy Server > Copy lại thông tin


Đối với iOS :

Vào thiết bị khác kết nối với Wifi > Dấu i > Định cấu hình Proxy > Thay đổi thông số > Lưu


Đối với Android :


Vào thiết bị khác kết nối với Wifi > Dấu i > Định cấu hình Proxy > Thay đổi thông số > Lưu


Đối với Windows :


Vào Setting > Network & Internet > Proxy > Manual proxy setup > Bật Use a proxy setup > Setting Address và Port > Save


Và thế là kết nối Wifi bình thường.
